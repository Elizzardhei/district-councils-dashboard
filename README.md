# District Councils Dashboard

> More transparency

District Councils Dashboard:
https://cswbrian.github.io/district-councils-dashboard

### Set-up

```bash
cd web

npm i

npm start
```

#### Convert people.csv to people.json

```bash
cd web/src/data

npm i

node csv2json
```
